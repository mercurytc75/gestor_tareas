🚀 CÓMO EMPEZAR (3 COMANDOS)
═════════════════════════════════════════════════════════════════════════════

1. Instala dependencias:
   $ pip install -r requirements.txt

2. Ejecuta la aplicación:
   $ python run.py

3. Abre en tu navegador:
   http://localhost:5000

🐛 SOLUCIÓN DE PROBLEMAS
═════════════════════════════════════════════════════════════════════════════

Problema: "No module named 'flask'"
→ Ejecuta: pip install -r requirements.txt

Problema: "Address already in use"
→ Cambia el puerto en run.py (línea 18)

Problema: "Database is locked"
→ Elimina instance/tareas.db y ejecuta nuevamente

Problema: "Los estilos no cargan"
→ Limpia caché: Ctrl+Shift+Supr (Chrome) o Ctrl+F5
